goblin_cbm_runner.cbm.data_processing.default_processing.AF_disturbances

This module is responsible for managing disturbances within a CBM (Carbon Budget Modeling) model.

It manages the creation of the disturbance input for legacy afforesation since 1990.

Manages disturbances within a CBM (Carbon Budget Model) model, addressing both legacy and scenario-based disturbances. This class plays a pivotal role in simulating the impact of disturbances on forest carbon stocks and fluxes, adapting to user-defined management strategies and afforestation scenarios.
