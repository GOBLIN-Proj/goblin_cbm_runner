goblin_cbm_runner.cbm.data_processing.geo_processing.geo_inventory

This module manages and processes forest inventory data for Carbon Budget Modeling (CBM) simulations at the catchment level, including handling of legacy forest data and afforestation projects. This class is essential for creating accurate and scenario-specific inventory structures that feed into CBM simulations, enabling detailed analysis of forest carbon dynamics over time.

  • delay (int, optional) – The delay in years for the inventory. Defaults to 0.
