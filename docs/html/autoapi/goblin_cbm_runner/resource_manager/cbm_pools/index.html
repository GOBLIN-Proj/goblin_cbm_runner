class goblin_cbm_runner.resource_manager.cbm_pools.Pools[source]

Class representing the pools in a CBM model.

This class encapsulates various types of pools relevant in the context of Carbon Budget Modeling (CBM), such as biomass pools, deadwood, litter, and soil organic matter. It also manages annual process fluxes.
