class cbm_methods.CBMSim

A class for running the Carbon Budget Model (CBM) and generating carbon stocks and fluxes for different scenarios.

This class provides functionalities to run the CBM model and generate carbon stocks and fluxes for different scenarios. The CBM model is a widely used model for estimating carbon stocks and fluxes in forest ecosystems. The model uses input data and parameters to simulate the carbon dynamics in forest ecosystems over time.
