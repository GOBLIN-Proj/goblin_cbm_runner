class cbm_runner.historic_affor.historic_affor_validation_manager.HistoricValidationManager[source]

This class is responsible for retrieving the validation data generated when running scenarios.

To generate validation data, the HistoricAfforRunner attribute must be set to `gen_validation` True and the valdiation path must be set.
